  • What does a website consist of?

    • JavaScript

      JavaScript is a programming language for adding interactivity and dynamic features to web pages, improving the user experience with real-time updates and animations. It runs in the user's browser.

    • HTML

      HTML (Hypertext Markup Language) is a markup language for creating web page structure and content. It uses tags to organize and make content readable in web browsers.

    • CSS

      CSS (Cascading Style Sheets) is used for styling web pages, controlling layout, colors, fonts, and design. It determines how content looks to users.
